## このアプリについて

「今期なに見る？」は視聴中のアニメを管理し、共有できるWebアプリケーションです。

Ruby on Rails 5.1で開発しています。

現在開発中ですが、Herokuにデプロイしてありますのでよろしければご覧ください。

https://konki-nanimiru.herokuapp.com/

## 個人メモ

データベースの起動

```
pg_ctl -l /usr/local/var/postgres/server.log start
```
